### 经典古风配色方案

#### 方案一：故宫红（庄重典雅）

适合：文化类、博物馆、高端品牌

```css
:root {
  --primary: #8B2500;       /* 朱砂红 */
  --primary-light: #C4564A; /* 浅朱 */
  --bg: #F5F0E8;            /* 米白 / 宣纸色 */
  --bg-secondary: #EDE5D8;  /* 深米白 */

  --text: #2C1810;          /* 深褐 */
  --text-secondary: #6B5B4E;/* 灰褐 */
  --border: #D4C5B0;        /* 浅驼色 */
}
```

#### 方案二：青瓷绿（清雅淡然）


#### 方案五：胭脂粉（温柔国风）

适合：女性向、汉服、美妆、婚庆

```css
:root {
  --primary: #C45A65;       /* 胭脂 */
  --primary-light: #E8959B; /* 浅胭脂 */
  --bg: #FDF6F0;            /* 粉白 */
  --bg-secondary: #F5EBE3;  /* 暖粉灰 */
  --accent: #D4A574;        /* 杏黄 */
  --text: #3C2A2A;          /* 深褐红 */
  --text-secondary: #8C6B6B;/* 灰粉 */
  --border: #E8D5C8;        /* 浅粉驼 */
}
```

### 配色规则

1. **色彩克制**：主色不超过 2 种，大量使用米白/象牙白底色
2. **禁用纯黑纯白**：用 `#2C1810`（深褐）代替 `#000`，用 `#F5F0E8`（米白）代替 `#FFF`
3. **金色点缀**：古铜金 `#C4956A` 用于重点元素（分割线、图标、价格）
4. **红色慎用**：朱砂红 `#8B2500` 只用于主操作按钮和品牌标识，不大面积铺开

### 排版方案

```css
/* 古风排版 */
font-family: "LXGW WenKai", "Noto Serif SC", "宋体", serif;

/* 字号层级 */
h1 { font-size: 36px; font-weight: 700; letter-spacing: 8px; }   /* 大标题 - 加宽字距 */
h2 { font-size: 28px; font-weight: 600; letter-spacing: 4px; }   /* 区块标题 */
h3 { font-size: 20px; font-weight: 500; letter-spacing: 2px; }   /* 子标题 */
body { font-size: 16px; font-weight: 400; line-height: 2.0; }    /* 正文 - 行高比一般网站更大 */
small { font-size: 14px; color: var(--text-secondary); }          /* 辅助文字 */
```

### 竖排文字

古风设计的独特排版方式：

## 五、布局与间距

### 古风留白原则

中国画讲究"计白当黑"，古风设计的间距比普通网站更大。

```css
/* 间距规范 */
--spacing-section: 96px;   /* 区块之间（普通网站 64px） */
--spacing-card: 24px;      /* 卡片内边距 */
--spacing-content: 48px;   /* 内容块之间 */

/* 内容区域 */
max-width: 960px;          /* 比一般网站窄（普通 1200px），更有呼吸感 */
margin: 0 auto;
padding: 0 24px;
```

### 布局模式

#### 1. 留白式布局（推荐）

```
┌──────────────────────────────────────┐
│                                      │
│            （大量留白）                │
│                                      │
│         ┌──────────────┐             │
│         │   标题       │             │
│         │   （竖排）    │             │
│         └──────────────┘             │
│                                      │
│    ┌────────────────────────┐        │
│    │   内容区域              │        │
│    │   max-width: 640px     │        │
│    └────────────────────────┘        │
│                                      │
│            （大量留白）                │
│                                      │
└──────────────────────────────────────┘
```

#### 2. 左右对称布局

```
┌──────────┬──────────────────┐
│          │                  │
│  竖排    │     主内容       │
│  导航    │     区域         │
│          │                  │
│  纹样    │                  │
│  装饰    │                  │
│          │                  │
└──────────┴──────────────────┘
```

#### 3. 卷轴式滚动

单页长滚动，模拟卷轴展开的体验：

```css
/* 卷轴过渡分割 */
.scroll-divider {
  height: 120px;
  background: url('cloud-pattern.svg') repeat-x center;
  opacity: 0.1;
}
```

## 六、微交互与动效

### 古风专属动效

```css
/* 水墨扩散效果（按钮 hover） */
.ink-button {
  position: relative;
  overflow: hidden;
  transition: color 0.5s ease;
}
.ink-button::before {
  content: '';
  position: absolute;
  width: 0; height: 0;
  border-radius: 50%;
  background: var(--primary);
  opacity: 0.15;
  transition: width 0.6s ease, height 0.6s ease;
}
.ink-button:hover::before {
  width: 300px; height: 300px;
  top: 50%; left: 50%;
  transform: translate(-50%, -50%);
}

/* 元素淡入 —— 比常规更慢，有禅意 */
.fade-in {
  opacity: 0;
  transform: translateY(30px);
  transition: all 0.8s cubic-bezier(0.25, 0.46, 0.45, 0.94);
}
.fade-in.visible {
  opacity: 1;
  transform: translateY(0);
}

/* 卡片 hover —— 微弱浮动 + 阴影加深 */
.card {
  transition: all 0.4s ease;
  box-shadow: 0 2px 8px rgba(44, 24, 16, 0.06);
}
.card:hover {
  transform: translateY(-3px);
  box-shadow: 0 8px 24px rgba(44, 24, 16, 0.12);
}
```

### 古风加载动画

```css
/* 旋转祥云加载 */
.loading-cloud {
  width: 40px; height: 40px;
  animation: float 2s ease-in-out infinite;
}
@keyframes float {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-10px); }
}
```

## 七、圆角与阴影

### 圆角规范

古风设计的圆角比一般网站略小，避免过于"现代感"：

```css
--radius-sm: 4px;    /* 按钮、输入框 */
--radius-md: 8px;    /* 卡片 */
--radius-lg: 12px;   /* 大容器、模态框 */
/* 不使用 16px+ 的大圆角，保持方正感 */
```

### 阴影规范

古风阴影偏暖色，不用纯黑/纯灰：

```css
--shadow-sm: 0 1px 3px rgba(44, 24, 16, 0.05);
--shadow-md: 0 4px 12px rgba(44, 24, 16, 0.08);
--shadow-lg: 0 12px 24px rgba(44, 24, 16, 0.12);
```

## 九、移动端适配

### App 设计要点

1. **底部导航**：图标用线性古风图标（iconfont 搜"古风图标"），不用 emoji
2. **状态栏**：深色内容页用白色状态栏文字
3. **启动页**：水墨动画或品牌纹样渐显
4. **空状态**：水墨风格插画 + 诗句文案（如"千山鸟飞绝"配空搜索结果）
5. **下拉刷新**：自定义为祥云或毛笔笔触动画

### 移动端间距

```css
/* 移动端间距略紧凑 */
--spacing-section-mobile: 64px;
--spacing-card-mobile: 16px;

/* 触摸目标 */
button, a { min-height: 44px; }
```

## 十、禁止清单

1. ❌ 使用 emoji 代替图标 → 用 iconfont 古风图标集
2. ❌ 纯黑 `#000` 或纯白 `#FFF` → 用深褐/米白
3. ❌ 默认蓝色 `blue-500` 作为主色 → 用朱砂红/青瓷绿等传统色
4. ❌ 大圆角 `16px+` → 保持方正感 `4-12px`
5. ❌ 冷灰色阴影 → 暖色调阴影 `rgba(44, 24, 16, ...)`
6. ❌ 无衬线字体做标题 → 宋体/楷体做标题
7. ❌ 渐变色按钮 → 纯色 + 纹样装饰
8. ❌ 现代感动效（弹跳、旋转） → 缓慢淡入、水墨扩散
9. ❌ 纹样原色直接使用 → 同色系低饱和处理
10. ❌ 普通分割线 `1px border` → 传统纹样分割线
